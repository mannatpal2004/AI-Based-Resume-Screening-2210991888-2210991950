## Abstract
This project applies Machine Learning and Natural Language Processing
to automate resume screening. Four classifiers (Logistic Regression,
SVM, Random Forest, Decision Tree) were trained on 1,000 resumes using
TF-IDF feature extraction. SVM achieved the best performance with
92.8% accuracy and 91.5% F1-score.
